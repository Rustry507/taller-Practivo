# taller-Practivo
Para ejecutar este proyecto es necesario tener instalado Python 3 y las librerías Flask y Flask-CORS
Una vez descargados los archivos del proyecto se debe abrir una terminal dentro de la carpeta donde se encuentran los archivos Taller.py WebPage.html presentacion.html y database.db.

Primero se instalan las dependencias ejecutando:
pip install flask pip install flask-cor 

Después se inicia la aplicación ejecutando:

python Taller.py

si el servidor inicia correctamente aparecerá un mensaje indicando que la aplicación se está ejecutando en http://127.0.0.1:5000.

y se debe abrir un navegador web y acceder a la siguiente dirección:

http://127.0.0.1:5000/

Desde ahí se mostrará la pantalla de inicio de sesión y despues de poner las credenciales usuario: admin contr: 1234 saldra el sistema de gestión de expedientes
